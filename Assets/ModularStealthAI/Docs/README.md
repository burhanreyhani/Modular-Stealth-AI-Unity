- Modular Stealth AI (Unity)

This is a free AI asset for Unity.  
It is designed as a customizable starting point for stealth games, prototypes, and learning purposes.

This asset contains:

- Enemy AI system with Patrol / Check / Chase / Search / Attack states
- Dynamic patrol point distribution system (randomized per enemy)
- Suspicion-based detection & alert system with smooth light feedback
- Global alarm system (all enemies react when one detects player)
- NavMesh-based movement & searching behavior
- Player detection using distance, angle and raycast visibility
- Demo level with:
  - Random key spawn system
  - Escape objective


Important! This asset has following dependencies:

- Unity Input System
  You must enable it from: 
  `Edit → Project Settings → Player → Active Input Handling → Input System Package`

- TextMeshPro is required for UI (included by default in most Unity projects).

- This asset uses Unity's NavMesh Surface and NavMesh Agent systems for enemy movement and pathfinding.

Controls:

- WASD / Arrow Keys – Controls
- R - Restart

Additional Notes:

- This asset do not have taking cover or crouching mechanichs.
- AI enemies passing through each other otherwise they might stuck and cannot move.

Intended Use:

- Learning stealth AI in Unity
- Prototyping stealth games
- Base system for further customization
- Beginner and Intermediate projects
